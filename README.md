# Retail Sales SQL Project

## Project Overview

This project analyzes retail sales data using PostgreSQL and SQL queries to generate business insights from customer transactions.

## Tools Used

* PostgreSQL
* pgAdmin
* SQL

## Dataset

Retail Sales Dataset (CSV)

## SQL Concepts Applied

* SELECT Statements
* WHERE Clause
* GROUP BY
* ORDER BY
* Aggregate Functions
* CASE Statements
* Common Table Expressions (CTEs)
* Date & Time Functions

## Business Problems Solved

1. Total Sales Analysis
2. Customer Purchase Analysis
3. Category-wise Performance
4. Shift-wise Order Analysis
5. Unique Customer Analysis
6. Revenue Insights

## Files Included

* retail_sales.sql
* SQL - Retail Sales Analysis_utf.csv
